- Before any playback starts pre-generate ALL audio files
- Call EdgeTTS Python script from Node via child_process for each text line

- Show progress bar with percentage: "Generating audio 45/300 lines"
- Measure exact duration of each audio file after generation
- Store all durations in a timeline array for sync
- Skip image lines — no audio needed
- If EdgeTTS fails on any line — skip it, log which line failed, continue
- Only proceed to playback after ALL audio is generated
